# 滕王阁序量化交易系统

## 已实现功能 ✅

### 数据层
- ✅ AkShare 免费实时行情接入（A股全市场）
- ✅ 价格缓存机制（5秒缓存，避免频繁请求）
- ✅ 自动降级（AkShare失败时使用模拟数据）

### 交易功能
- ✅ 持仓扫描（止损/止盈/时间止损自动检测）
- ✅ **一键卖出**（批量处理信号，交互式确认）
- ✅ **手动卖出**（指定股票代码卖出）
- ✅ **交易记录自动保存**（JSON格式，按月归档）
- ✅ **交易统计**（胜率、盈亏比、笔均盈亏）

### 通知系统
- ✅ 钉钉推送
- ✅ 飞书推送
- ✅ 控制台实时显示

### 定时任务（已配置）
- 09:00 盘前扫描
- 13:00 午盘扫描  
- 15:30 收盘扫描

## 快速开始

### 1. 安装依赖

```bash
cd /root/.openclaw/workspace/quant
pip install -r requirements.txt
```

### 2. 运行扫描（检查信号）

```bash
python3 engine.py
```

### 3. 进入交互模式（交易操作）

```bash
python3 engine.py --interactive
```

交互模式菜单：
```
🏮 滕王阁序交易系统 - 交互模式
============================================================
1. 扫描持仓 (检查止损/止盈/时间止损)
2. 一键卖出 (批量处理信号)
3. 手动卖出 (指定股票)
4. 查看交易记录
5. 生成报表
q. 退出
```

## 交易操作流程

### 场景1：一键卖出所有触发信号的股票

```bash

# 选择 1 (扫描)
# 选择 2 (一键卖出)
# 输入 a (全部卖出) 或 1,2,3 (选择卖出)
```

### 场景2：手动卖出指定股票

```bash
python3 engine.py --interactive
```

## 数据来源

当前使用 **AkShare** 免费数据源：
- A股实时行情（东方财富）
- 每5秒刷新一次缓存
- 无需API Key

如需更稳定数据，可切换至：
- Tushare（需注册Token）
- 同花顺iFinD（付费）

---

## 注意事项

1. **当前为模拟交易**：卖出操作只更新本地配置，不涉及真实资金
2. **时间止损已生效**：系统会自动检测超过 `max_hold_days` 的持仓
3. **交易记录持久化**：所有交易自动保存到 `trades/` 目录
4. **定时任务已配置**：交易日自动扫描三次
